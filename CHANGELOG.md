# Changelog

## 1.0.0 — 2026-04-19

Initial release.

- 8-mode skill dispatcher (`default`, `--quick`, `--kapil`, `--debate`, `--pushback`, `--retro`, `--eval`, `--context`)
- Brain synthesized from 34 long-form Naval Ravikant interviews via 6 parallel Opus extraction agents + 1 synthesis pass
- 8 canonical regression-test Q&A pairs in `_evals/canonical.md`
- Portable install: bundled brain, env-var overrides, no vault dependency
- Korean triggers (`네이벌`, `카필`, etc.) in addition to English
- MIT license with explicit non-endorsement clause
